Detailed Description

Time Plugin Implementation.

Version
1.0.0

Provides current date/time in various formats:

  • "now" or empty: Full ISO timestamp
  • "date": Date only (YYYY-MM-DD)
  • "time": Time only (HH:MM:SS)
  • "unix": Unix timestamp
