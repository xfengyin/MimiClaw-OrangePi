Process a user message and generate a response.

Sends the user message to the AI model and returns the response. Handles tool calls automatically if the model requests them.

Parameters
ctxCore context (must be initialized)
session_idUnique session identifier for conversation history
user_messageUser's input message
responsePointer to receive the response structure
Returns
MIMI_CORE_OK on success, error code on failure
Note
Memory: Caller must free response->content and response->finish_reason
Session: Creates session automatically if it doesn't exist

◆ mimi_core_init()

int mimi_core_init ( mimi_core_ctx_t **  ctx,
const mimi_core_config_t *  config 
)

Initialize a MimiCore context.

Allocates and initializes a new MimiCore context with the provided configuration. The context must be destroyed with mimi_core_destroy() when no longer needed.

Parameters
ctxPointer to context pointer to be initialized
configConfiguration structure (must be valid)
Returns
MIMI_CORE_OK on success, error code on failure
Note
Thread-safe: Each context is independent and can be used in separate threads
Memory: Caller must call mimi_core_destroy() to free resources
