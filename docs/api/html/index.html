📚 libmimi-core API 详解

int mimi_core_init(const char* config_path)

初始化 MimiClaw 核心库，加载配置文件并初始化数据库连接。

参数: config_path - 配置文件路径 (JSON/YAML)

返回: 0 表示成功，非 0 表示错误码

// 初始化核心库 int ret = mimi_core_init("./config/mimiclaw.json"); if (ret != 0) { fprintf(stderr, "初始化失败：%d\n", ret); }

session_t* mimi_session_create(const char* user_id, const char* agent_id)

创建新的会话，返回会话句柄。

参数: user_id - 用户 ID, agent_id - Agent ID

返回: 会话句柄，失败返回 NULL

// 创建会话 session_t* sess = mimi_session_create("user_001", "dev-planner"); if (sess) { printf("会话创建成功：%p\n", (void*)sess); }

int mimi_message_send(session_t* sess, const char* content, message_type_t type)

向会话发送消息。

参数: sess - 会话句柄, content - 消息内容, type - 消息类型

返回: 消息 ID，失败返回 -1

// 发送消息 int msg_id = mimi_message_send(sess, "你好，请帮我规划项目", MSG_TYPE_USER); printf("消息 ID: %d\n", msg_id);

void mimi_core_cleanup(void)

清理核心库资源，关闭数据库连接，释放内存。

// 程序退出前清理 mimi_core_cleanup();

📚 libmimi-memory API 详解

mempool_t* mimi_mempool_create(size_t block_size, size_t block_count)

创建内存池，预分配指定数量和大小的内存块。

参数: block_size - 每块大小 (字节), block_count - 块数量

返回: 内存池句柄，失败返回 NULL

// 创建 4KB x 256 块的内存池 mempool_t* pool = mimi_mempool_create(4096, 256); if (pool) { printf("内存池创建成功\n"); }

void* mimi_mempool_alloc(mempool_t* pool, size_t size)

从内存池分配内存。

参数: pool - 内存池句柄, size - 请求大小

返回: 分配的内存指针，失败返回 NULL

// 从内存池分配 void* buf = mimi_mempool_alloc(pool, 1024); if (buf) { memset(buf, 0, 1024); }

void mimi_mempool_free(mempool_t* pool, void* ptr)

释放内存回内存池。

参数: pool - 内存池句柄, ptr - 要释放的指针

// 释放内存 mimi_mempool_free(pool, buf);

int mimi_mempool_stats(mempool_t* pool, mempool_stats_t* stats)

获取内存池统计信息。

参数: pool - 内存池句柄, stats - 统计结构体指针

返回: 0 表示成功

// 获取统计 mempool_stats_t stats; mimi_mempool_stats(pool, &stats); printf("已使用：%zu / %zu\n", stats.used, stats.total);
