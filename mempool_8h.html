Detailed Description

MimiClaw Memory Pool - Efficient Block Allocation.

Version
2.0.0
Date
2026-03-10
Copyright
Copyright (c) 2026 MimiClaw Project @license MIT License

内存池实现，用于减少 malloc/free 碎片，降低内存占用

使用场景:

  • 消息缓冲区 (4KB)
  • 会话上下文 (2KB)
  • HTTP 响应缓冲 (16KB)
