# ============================================================================
# Media-Creator 心跳任务
# ============================================================================

## 定时检查 (每 30 分钟)

### 1. 社区互动检查
- 检查 GitHub Issues 新增问题
- 检查是否有用户提问需要回答
- 检查社交媒体提及

### 2. 内容发布检查
- 检查是否有待发布的版本
- 检查 CHANGELOG 更新
- 检查博客/教程进度

### 3. 反馈收集检查
- 整理用户反馈
- 分类: Bug 报告、功能请求、文档问题
- 转达给 Dev-Planner
