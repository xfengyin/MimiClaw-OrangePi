# Dev-Coder 心跳任务
# ============================================================================

## 定时检查 (每 10 分钟)

### 1. 构建状态检查
- 检查 `make -C libs` 是否通过
- 检查 `make -C plugins` 是否通过
- 失败则定位错误模块

### 2. 测试结果检查
- 检查 `make -C libs test` 测试结果
- 记录失败测试用例
- 统计覆盖率变化

### 3. 代码质量检查
- 检查是否有未提交的更改
- 检查编译警告数量
- 检查是否有 TODO/FIXME 标记

### 4. 任务队列检查
- 检查 Dev-Planner 分配的新任务
- 检查 Bug 修复队列
- 优先级: 阻塞 Bug > 功能开发
